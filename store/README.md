# Bienvenido a tu aplicación de Expo

Este es un proyecto de [Expo](https://expo.dev) creado con [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Empecemos

1. Instalar dependencias

   ```bash
   npm install
   ```

2. Iniciar la aplicación

   ```bash
    npx expo start
   ```

En la consola de salida, encontrarás opciones para abrir la aplicación en:

- [Construcción de desarrollo](https://docs.expo.dev/develop/development-builds/introduction/)
- [Emulador de Android](https://docs.expo.dev/workflow/android-studio-emulator/)
- [Simulador de iOS](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), un entorno de pruebas limitado para probar el desarrollo de aplicaciones con Expo.

Puedes empezar a desarrollar editando los archivos dentro del directorio **app**. Este proyecto utiliza [enrutamiento basado en archivos](https://docs.expo.dev/router/introduction).

## Reiniciar el proyecto

Cuando estés listo para un proyecto limpio, ejecuta:

```bash
npm run reset-project
```

Este comando moverá el código inicial al directorio **app-example** y creará un directorio **app** vacío donde podrás empezar a desarrollar.
